Introduction

Loops are one of the most important concepts in the C programming language. They are used to execute a block of code multiple times based on a given condition. Instead of writing the same lines again and again, loops help reduce redundancy and make the code cleaner and easier to manage.

Using loops, programmers can create efficient solutions for tasks like printing sequences, calculating values repeatedly, processing arrays, and automating repetitive actions. Without loops, programmers would need to manually write each step, which is time-consuming and inefficient.

C supports three types of loops: while, for, and do-while. Each loop has its own structure and use case. This project explains each type in simple language, shows example programs, and includes a comparison table to help understand their differences clearly.

While Loop

The while loop checks the condition before executing the code block. If the condition is true, the loop runs.

while (condition) {
  // code to execute
}

int i = 1;
while (i <= 5) {
  printf("%d\n", i);
  i++;
}

For Loop

The for loop is used when the number of iterations is known.It has initialization, condition, and update expressions.

for (initialization; condition; update) {
// code to execute

}

for (int i = 1; i <= 5; i++) { 
	printf("%d\n", i); 
}

Do-While Loop

The do-while loop executes the code block once before checking the condition.

do {
  // code to execute
} while (condition);

int i = 1;
do {
  printf("%d\n", i);
  i++;
} while (i <= 5);
